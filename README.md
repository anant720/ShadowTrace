# ShadowTrace: Intelligence-Driven Threat Forensics v6.0

ShadowTrace is a state-of-the-art security intelligence platform designed to neutralize phishing, credential exfiltration, and adversarial URL attacks in real-time. Built for the Modern Enterprise, it combines advanced Machine Learning ensemble models with high-fidelity network forensics to protect against sophisticated social engineering.

## 🚀 Key Enterprise Capabilities

### 🧠 Multi-Layer AI/ML Intelligence
- **Ensemble Scorer**: Orchestrates Gradient Boosting (XGBoost) and Unsupervised Isolation Forests for high-precision detection.
- **Deep Packet Analysis**: Real-time evaluation of entropy, homographs, and obfuscated JS indices.
- **Explainable AI (XAI)**: Full transparency on risk factors for every analysis event.

### 🛡️ Passive Forensic Audit Suite
- **Infrastructure Privacy Shield**: Automatically excludes internal organizational traffic to prevent credential leakage.
- **Real-Time Data Redaction**: Server-side scrubbing of captured JWTs, passwords, and API keys before persistent storage.
- **Deep Forensic Inspector**: Captures full-spectrum request/response payloads with sensitive data masking in the UI.

### 🆔 User Attribution & Accountability
- **Gmail Identity Association**: Every forensic scan is correlated with the specific user's Gmail ID via `chrome.identity`.
- **Sovereignty Tracking**: Tracks which organizational asset was targeted, enabling rapid incident response.

### 🖱️ On-Demand Privacy Model
- **Opt-In Protection**: Zero auto-scanning. Analysis only occurs when an analyst explicitly triggers a scan via the extension.
- **Silent Mode**: Zero background footprint until activated, ensuring analyst privacy and performance.

## 📊 Command & Control Dashboard
- **Universal Responsive UI**: Seamlessly manage security intelligence across Desktop, Tablet, and Mobile.
- **Threat Concentration Matrix**: Automated identification of high-risk domains and adversarial TLDs.
- **Security Posture Analytics**: Global visualization of organizational risk distribution.
